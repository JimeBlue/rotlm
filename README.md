# Raiders of the Lost Missile

Band website for Raiders of the Lost Missile, built with Nuxt 4.

## Tech Stack

- **Frontend**: Nuxt 4, Vue 3, TypeScript
- **Styling**: Tailwind CSS v4, Nuxt UI
- **CMS**: Sanity (content + Sanity Studio)
- **Email**: Resend
- **Backend**: Nitro server routes
- **Icons**: nuxt-svgo
- **Animation**: VueUse Motion
- **Internationalization**: EN, DE, IT, ES (default: DE)
- **Validation**: Yup

## Getting Started

```bash
# Install dependencies
yarn install

# Start development server
yarn dev
```

## Sanity Studio

The Sanity Studio is located in the `studio/` directory.
